## Useful idea

A personal AI command center should separate actions into clear safety levels:

- **Read-only:** status checks, summaries, weather, repository checks.
- **Low-risk automatic:** reminders, daily notes, safe notifications.
- **Approval-required:** posting, buying, deleting, booking, smart-home device control.
- **Never public:** tokens, OAuth files, private documents, chat logs, exact device credentials.
