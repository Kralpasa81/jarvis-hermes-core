## Gelecekteki Modül Fikri
**Modül: `config_drift_detector`**
İki farklı ortamın (örn. `config/dev.yaml` ve `config/prod.yaml`) şema
yapısını karşılaştırıp "sürükleme" (drift) olan alanları raporlayan araç.
Gerçek değerleri karşılaştırmaz — sadece anahtar adları ve tipleri.
Böylece prod config'de beklenmedik alan eksikliği veya tip değişikliği
erken tespit edilir. Token/API key gerektirmez.

## Güvenlik / Secret Hatırlatması
- Bağımlılık tanımlarını YAML/JSON dosyasından okurken yol traversal
  kontrolü yap: `../` içeren modül adlarını reddet.
- Mermaid çıktısında modül adları kullanıcı verisinden geliyorsa
  `<`, `>`, `"` karakterlerini temizle; GitHub render'ında XSS riski oluşmaz
  ancak diyagram bozulmasını önler.
- Commit öncesi `git diff --stat` ile eklenen dosyaları gözden geçir.
